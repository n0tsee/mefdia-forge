# Установка MediaForge (macOS)

Файл: **`MediaForge_0.1.0_aarch64.dmg`** (~61 МБ)
Подходит для Mac на **Apple Silicon** (M1/M2/M3/M4).

## Шаги

1. Открыть `MediaForge_0.1.0_aarch64.dmg` двойным кликом.
2. Перетащить **MediaForge** в папку **Applications**.
3. Извлечь образ (кнопка ⏏ рядом с диском в Finder).

## ⚠️ Первый запуск — важно

Приложение подписано «для себя» (без платного сертификата Apple за $99/год),
поэтому при скачивании из интернета/мессенджера macOS ставит «карантин» и
покажет что-то вроде **«MediaForge повреждён»** или **«не удаётся открыть,
неизвестный разработчик»**. Это НЕ вирус — так Gatekeeper реагирует на любое
неподписанное Apple приложение.

**Как разрешить запуск — один раз:**

Открыть **Терминал** (Программы → Утилиты → Терминал) и вставить команду:

```bash
xattr -cr /Applications/MediaForge.app
```

Нажать Enter. После этого приложение открывается обычным двойным кликом.

> Если перетащили не в Applications, а, например, на Рабочий стол — укажите
> в команде свой путь, например: `xattr -cr ~/Desktop/MediaForge.app`

## Альтернатива без Терминала

1. Правый клик по `MediaForge.app` → **Открыть**.
2. В диалоге ещё раз нажать **Открыть**.

(На новых версиях macOS этот способ иногда не срабатывает для неподписанных
приложений — тогда используйте команду `xattr` выше, она работает всегда.)

## Что внутри

FFmpeg уже встроен в приложение — ничего дополнительно ставить не нужно.
Приложение полностью оффлайн.

---

### Чтобы совсем без предупреждений (для автора)

Единственный способ убрать предупреждения Gatekeeper полностью — подписать
приложение сертификатом **Apple Developer ID** и пройти **нотаризацию** у Apple.
Для этого нужен платный аккаунт Apple Developer ($99/год). После этого `.dmg`
будет открываться двойным кликом у любого пользователя без команд в терминале.
